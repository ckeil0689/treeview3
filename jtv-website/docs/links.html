Java Treeview Documentation: Url Link Templates

Java Treeview is shipped with a set of link templates, available under "Settings->Url Settings...", for common databases. However, there are many more databases available.

To use these templates, either paste them into the "Settings->Url Settings" dialog, or if you are likely to use a particular one often, go to "Settings->Presets->Gene Url Presets" and make a preset for it.

Stock Tempates

Name Template
SGD http://genome-www4.stanford.edu/cgi-bin/SGD/locus.pl?locus=HEADER
WormBase http://www.wormbase.org/db/searches/basic?class=AnyGene&query=HEADER&Search=Search
Source_CloneID (same as SUID?) http://genome-www4.stanford.edu/cgi-bin/SMD/source/sourceResult?option=CloneID&choice=Gene&criteria=HEADER
FlyBasehttp://flybase.bio.indiana.edu/.bin/fbgenq.html?HEADER
MouseGD http://www.informatics.jax.org/javawi/servlet/SearchTool?query=HEADER&selectedQuery=Genes+and+Markers
GenomeNetEcoli http://www.genome.ad.jp/dbget-bin/www_bget?eco:HEADER

When the template is used, the text "HEADER" is filled in, and a browser window is opened with the resulting URL.
